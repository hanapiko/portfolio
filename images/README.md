# Images Folder Structure

## Folder Organization:
- `banking/` - Mobile Banking App project images
- `food/` - Food Delivery Platform project images  
- `fitness/` - Fitness Tracker Redesign project images

## Image Naming Convention:

### Banking App (`banking/` folder):
- `banking-cover.jpg` - Cover image (400x300px) for project card
- `banking-1.jpg` - Overview/main screen
- `banking-2.jpg` - Login screen design
- `banking-3.jpg` - Dashboard interface
- `banking-demo.mp4` - App demo video (optional)

### Food Delivery (`food/` folder):
- `food-cover.jpg` - Cover image (400x300px) for project card
- `food-1.jpg` - App overview
- `food-2.jpg` - Menu interface
- `food-3.jpg` - Order tracking screen

### Fitness Tracker (`fitness/` folder):
- `fitness-cover.jpg` - Cover image (400x300px) for project card
- `fitness-1.jpg` - App overview
- `fitness-2.jpg` - Workout tracking
- `fitness-3.jpg` - Progress analytics

## Image Specifications:
- **Cover images**: 400x300px (for project cards)
- **Gallery images**: 800x600px or higher (for modal display)
- **Format**: JPG, PNG, or WebP
- **Videos**: MP4 format recommended

## Tips:
- Use high-quality images that showcase your design work
- Ensure consistent lighting and presentation style
- Consider using device mockups for mobile app designs
- Keep file sizes reasonable for web performance
